## 🚀 Featured Projects

### 🔐 [Post-Quantum Secure Communication System](https://github.com/ramangandewar) · *Barclays Sponsored*
> Quantum-resilient cryptographic migration framework using **ML-KEM + ML-DSA**, implementing a 5-state Structured Migration State Machine (SMSM). Built an HNDL risk scorer, React dashboard, Prometheus metrics, and Docker deployment. Targeting **IEEE S&P / ACM CCS** publication.

### 🎭 [FrameTruth: Deepfake Video Forensics Platform](https://github.com/ramangandewar) · *SVL Technologies Sponsored*
> Production-grade deepfake detection with ensemble **TFLite inference**, temporal consistency checks, optical-flow, and frequency-domain forensic signals. JWT + RBAC auth, async job processing, Prometheus/Grafana observability, and exportable PDF forensic reports with per-frame confidence scores.

### 📄 [Tender Proposal Automation Platform](https://github.com/ramangandewar) · *DataSmith AI Sponsored*
> **LangGraph-orchestrated** pipeline that parses tender PDFs, retrieves company knowledge via Qdrant vector search, and generates structured DOCX proposals. Includes vendor RFQ via SMTP, Cohere rerank compliance scoring, and Supabase-backed orchestration persistence.

### 🤖 RAG-Based Intelligent Assistant · *Renishaw Sponsored*
> Enterprise chatbot using **FAISS vector search + LLM integration** over a multi-source knowledge base. Deployed via Streamlit for internal IT domain queries. Handles context-aware retrieval across diverse documentation sources.

### 🏥 HealthGuard-FL *(Research)*
> Privacy-first **Federated Learning** framework for cross-institutional sepsis prevention. Achieves AUC >0.86 with ϵ≈0.11 differential privacy. **Scopus-indexed, 2025.**
